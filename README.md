
## Trip-Memories

* Ruby version - ruby 2.4.0

## MVP
- [X] Create User Account 
- [X] Log in / Log out
- [X] User can create a trip 
- [X] User can add posts to a trip
- [X] Models - User, Trip & Post has CRUD functions

## Description

Trip Memories allows you to save your highlights by creating posts with photos & text for that particular trip.

## Technology Stack
```
Ruby on Rails
PostgreSQL
Javascript / AJAX / JSON
Bootstrap
Heroku
PaperClip
Google Maps api
AWS - S3 Services
SASS
```

## To Run Locally
```
# Install Gem Files
bundle install
```

```
# Run the Server
rails s
```
